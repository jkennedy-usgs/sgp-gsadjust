3. Data tab

 
The data tab provides functions for viewing samples at each station occupation and tools for selecting/deselecting samples based on quality criteria.
 

Data Tree View

  The Data Tree View shows the hierarchical structure of the data set, and is common to all tabs.

Data Plot Window

The Data Plot window shows graphically the gravity value and accompanying data for a particular station occupation. For the Burris meter, the plots shown are gravity, feedback, tilt correction, and Earth tide correction. For the Scintrex meter, the plots shows are gravity, Tilt X, Tilt Y, and standard deviation.

check selected

Rows in the Data Table can be selected using the mouse; multiple rows can be selected by holding down the shift or control keys. If unchecked rows are selected, this button checks them.
Top

check all

Checks all rows in the Data Table.
Top

uncheck selected

Rows in the Data Table can be selected using the mouse; multiple rows can be selected by holding down the shift or control keys. If checked rows are selected, this button unchecks them.
Top

uncheck all

Unchecks all rows in the Data Table.
Top

auto uncheck |tilt| >

Unchecks observations with abs(tilt) (X or Y) greater than the specified value. Used only for Scintrex data.
Top

auto uncheck SD >

Unchecks observations with standard deviations greater than the specified value. Used only for Scintrex data.
Top

auto uncheck g >

Unchecks observations with g values that differ from the mean by more than the specified value.
Top

auto uncheck dur >

Unchecks observations with durations longer than the specified value. Used only for Scintrex data.
Top

apply all filters

Applies any thresholds for tilt, g, standard deviation, or duration to all samples of the current station occupation.
Top

apply all filters to all data

Applies any thresholds for tilt, g, standard deviation, or duration to all stations in the Campaign.
 
WARNING: This can be very slow for large datasets.
